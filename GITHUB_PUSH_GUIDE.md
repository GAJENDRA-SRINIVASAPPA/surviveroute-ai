## 🔧 Troubleshooting

### Error: "Repository not found"
**Solution:** You haven't created the repository on GitHub yet. Go to Step 1.

### Error: "remote origin already exists"
**Solution:** Run this instead:
```bash
git remote set-url origin https://github.com/GAJENDRA-SRINIVASAPPA/surviveroute-ai.git
git push -u origin main
```

### Error: "Authentication failed"
**Solution:** You need to authenticate with GitHub:
1. GitHub may ask for username/password
2. Use your GitHub username
3. For password, use a **Personal Access Token** (not your GitHub password)
4. Create token at: https://github.com/settings/tokens
